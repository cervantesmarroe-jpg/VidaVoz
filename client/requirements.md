## Packages
zustand | Lightweight state management for WebGazer and TTS settings

## Notes
- WebGazer.js is loaded dynamically from an external script (https://webgazer.cs.brown.edu/webgazer.js) to avoid package build issues and ensure compatibility with the browser's camera.
- The Web Speech API is natively supported in modern browsers and does not require an external package.
- The UI strictly enforces Dark Mode for high contrast and reduced eye strain in ICU environments.
